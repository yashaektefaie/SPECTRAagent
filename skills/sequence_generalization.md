# Sequence Generalization Skill

Use this skill for DNA, RNA, protein, and variant-effect prediction benchmarks.

Common spectral properties:

- Sequence identity.
- Edit or Hamming distance.
- Mutational-position distance.
- Motif disruption similarity.
- Regulatory class or assay family.
- Embedding distance.

Required controls:

- Distinguish interpolation among nearby variants from extrapolation to unseen regions.
- Check compositional confounders such as GC content when relevant.
- Report whether performance changes with measured sequence novelty.
