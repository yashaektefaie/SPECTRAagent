# Scientific Shift Generalization Skill

Use this skill for non-biology AI4Science benchmarks with explicit scientific
distribution shifts.

Common spectral properties:

- Covariate shift distance.
- Conditional shift distance.
- Concept or target-regime distance.
- Geometry or graph-feature similarity.
- Simulation, instrument, or environment context.

Required controls:

- Define the scientific unit of generalization before choosing a graph.
- Report which shift category is audited.
- Avoid treating benchmark labels as the spectral property unless the scientific
  question is target extrapolation.
