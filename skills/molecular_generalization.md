# Molecular Generalization Skill

Use this skill for molecule-level property prediction, virtual screening, and
drug-discovery benchmarks.

Common spectral properties:

- Morgan fingerprint Tanimoto similarity.
- Scaffold or chemotype similarity.
- Molecular embedding distance.
- Property-value extrapolation distance.
- Assay or cell-line context similarity.

Required controls:

- Check whether scaffold splits still leave high fingerprint similarity.
- Report whether performance degrades with chemical novelty.
- Separate chemical novelty from label imbalance and assay shift.
